## Featured Projects

### [WindLoader Calculator](https://github.com/Escgot)
> Structural wind load analysis software for civil engineers, built to Eurocode 1 (EN 1991-1-4) standards.

Full-stack engineering application with a FastAPI computational backend, 3D load visualization, and automated PDF report generation. Designed to take a structural problem from input parameters to a client-ready engineering report.

**Stack:** `Python` `FastAPI` `Scientific Computing` `3D Visualization` `Report Automation`

---

### [Reinforced Concrete Beam Designer](https://github.com/Escgot)
> RC beam design tool implementing Eurocode 2 (EN 1992-1-1) for structural engineers.

End-to-end application with a Python calculation engine, modern React interface, and Docker-based deployment. Handles section design, reinforcement layout, and compliance checking against EC2 requirements.

**Stack:** `Python` `FastAPI` `React` `TailwindCSS` `Docker`
